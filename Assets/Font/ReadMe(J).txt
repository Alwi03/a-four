◆FONT FORMAT◆1 BYTE FONTPostScript Type1 Font (For Macintosh)TrueType Font (For Windows)◆動作環境◆Macintosh : 漢字Talk7.51以上、Adobe Type Manager4.0以上Windows : Windows95以上、True Type Fontが使用可能な環境。◆使用方法◆Macintosh/すべてのアプリケーションを終了させ、システムフォルダ内のフォントフォルダに入れて使用してください。（スーツケースファイルとプリンタフォントファイルの2つを入れて使用してください。）Windows/「マイコンピュータ」→「コントロールパネル」→「フォント」の順で開く。「ファイルメニュー」から「新しいフォントのインストール」を選択。インストールしたいフォントファイルが入っているフォルダを選ぶ。フォントの一覧にインストール出来るフォントが表示されるので、そこからインストールしたいフォントファイルを選択しOKをクリック。以上でインストール終了です。◆キー配列◆Macintosh/アップルメニューの「キー配列」を参考にして入力してください。ひらがな・カタカナフォントは、基本的に濁音・小文字は+shift、半濁音は+optionです。例外として「づ・ヅ」、「げ・ゲ」は+option、等があります。Windows/従来の1 Byteひらがな・カタカナフォントの様なAlt key,Code numberは使用しません。小文字、他一部の文字は+shiftで入力します。濁音・半濁音記号は独立しています。Macintosh,Windows共に必ず「半角英字」で入力してください。通常の日本語入力変換は出来ません。すべてのフォントの著作権は石川和好(KAZUYOSHI ISHIKAWA)に属します。基本的には自由に使用して構いません。商用等で使用される場合は必ず連絡してください。著作者に無断での（改名・改造を含む）フォントの販売・配布・掲載・転載等の行為を堅く禁じます。フォントの使用によるいかなる損害に対して、当方は一切の責任を負いません。Copyright(C)2002 Kazuyoshi Ishikawa.
