This fonts was made by Japanese OS. If you use other OS, please use font installation of your OS.TrueType Font Installation For Windows (Windows 95 Later)1. From the Control Panel open the Fonts folder. 2. From the File menu, click Install New Font... 3. Choose the appropriate drive and directory that contain the font you want to add. 4. Select the fonts you wish to install and click the OK button. 5. Click the Cancel button. This font's copyright belong to Kazuyoshi Ishikawa.If you use this font on business, please send a mail to meYou can not sell, publish and distribute.Copyright(c)2002 KAZUYOSHI ISHIKAWA.
